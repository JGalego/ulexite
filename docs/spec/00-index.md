# Ulexite Language Specification (RFC-0001)

**Status:** Draft
**CLI:** `ulx`

Ulexite is a programming language whose primary abstraction is the **conversation** — not the prompt, the model, or the agent. It compiles conversations involving humans, LLMs, tools, judges, datasets, and multimodal artifacts to a deterministic execution graph, with reproducible traces and first-class testing as language features, not bolted-on libraries.
